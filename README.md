## Formspree
İletişim formu Formspree ile çalışır. `iletisim.html` içinde şu satırı kendi Formspree form ID’nize göre güncelleyebilirsiniz:

`action="https://formspree.io/f/XXXXXXX"`

## Yerel SEO (Mersin odaklı)
Sayfalara `MovingCompany` JSON-LD schema eklendi ve içerik Mersin/Akdeniz odaklıdır.

## Çalıştırma
- VS Code ile klasörü açın.
- `index.html` dosyasını Live Server ile çalıştırın (önerilir).

## Deploy (Vercel)
- Repo’yu GitHub’a gönderin.
- Vercel > New Project > repo seçin.
- Framework seçmeden “Other / Static” olarak deploy edebilirsiniz.
